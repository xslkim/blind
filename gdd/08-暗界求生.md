## 1. 游戏概述

### 1.1 核心信息

| 项目 | 内容 |
|------|------|
| 游戏名称 | 暗界求生（Dark Survival） |
| 类型 | 音频 Roguelike 生存 |
| 平台 | iOS / Android |
| 技术栈 | Unity + Resonance Audio |
| 目标受众 | Roguelike 玩家、生存游戏爱好者、恐怖游戏玩家 |
| 游戏时长 | 单局 20-45 分钟，全解锁 20+ 小时 |
| 定价模式 | 买断制 $6.99 |

### 1.2 Elevator Pitch

> 世界陷入永久黑暗后的第 100 天。你看不见任何东西——但你能听见它们在靠近。《暗界求生》是一款 Roguelike 生存游戏，在绝对黑暗中靠听觉收集资源、对抗怪物、寻找幸存者。每次死亡都是新的开始，每次重生你都能"听到"更多。

### 1.3 核心体验目标

- **紧张与恐惧**：每一步都可能触发危险，每个声音都可能是敌人
- **成长的正反馈**：死亡不可怕，每次重生后玩家变强的感觉很明显
- **随机性的乐趣**：每一局的地图、资源、生物组合都不同

## 2. 核心玩法

### 2.1 游戏循环

```
生成随机地图 → 探索房间 → 判断环境+收集资源 → 遭遇敌人 → 战斗或躲避
       ↑                                                        ↓
       └── 死亡 → 结算 → 永久升级 → 重生（新地图）←──────────┘
```

### 2.2 核心机制

#### 2.2.1 房间声学探索

进入新房间后，利用环境声音判断房间的性质：

| 声音线索 | 可推断信息 | 可靠度 |
|---------|-----------|--------|
| 拍手回声 | 房间大小（大/中/小/走廊） | 高 |
| 脚踢地面声 | 材质（混凝土/木头/泥土/金属） | 高 |
| 风声 | 是否有窗户或通风口 | 中 |
| 滴水声 | 水源在附近 | 中 |
| 机械声 | 有正在运转的设备 | 高 |
| 低语声 | 有 NPC 或"呢喃者"怪物 | 低（需确认） |
| 完全的寂静 | 安全区或"静默猎手"的狩猎区 | 低（危险） |

#### 2.2.2 怪物生态

| 怪物 | 标志声音 | 攻击模式 | 弱点 |
|------|---------|---------|------|
| 呢喃者 | 人类的低语，靠近时 UI 提示音扭曲 | 中速追猎，攻击前低语变为尖叫 | 静默移动可避开 |
| 咔嗒兽 | 骨节碰撞的咔嗒声，移动极快 | 高速冲撞，冲撞前咔嗒声频率翻倍 | 侧向闪避 |
| 尖叫行者 | 周期性发出尖叫，自身也靠声呐导航 | 追踪声源，但完全静止时它无法定位 | 屏息静止 |
| 盲猎者 | 完全沉默，唯一线索是偶尔带起的石子声 | 埋伏型，一击致命 | 提前听到石子声并避开 |
| 肿胀体 | 低沉的咕噜声，体积巨大 | 堵住关键通道，不主动攻击 | 用声音引诱移开 |
| 回声兽 | 模仿玩家发出的声音并回声回来 | 混淆声呐判断，误导玩家 | 识别声音的细微失真 |

#### 2.2.3 资源系统

| 资源 | 声音标识 | 用途 |
|------|---------|------|
| 饮用水 | 液体晃动声 | 恢复耐力，口渴度管理 |
| 食物罐头 | 金属碰撞声 + 塑料包装摩擦声 | 恢复血量，饥饿度管理 |
| 绷带 | 布料撕裂声 | 止血，移除流血状态 |
| 电池 | 拿起来有微弱电流声 | 为手电筒供电（虽然无光，但手电筒可以当作"声波枪"） |
| 武器零件 | 金属零件碰撞声 | 制作和升级武器 |
| 录音带 | 磁带旋转声 | 解锁背景故事碎片 |
| 信号弹 | 火药摩擦声 | 发射信号弹发出巨大声响，吸引/驱散怪物 |

#### 2.2.4 幸存者系统

在地图中随机遇到其他幸存者（NPC），每个人有不同的声音特征和需求：

| 幸存者 | 声音特征 | 需求 | 加入后的增益 |
|--------|---------|------|------------|
| 老兵 | 低沉的男声，断断续续 | 需要武器 | 近战伤害 +20% |
| 医学生 | 年轻的紧张女声 | 需要药物 | 包扎效果翻倍 |
| 工程师 | 语速快，自言自语 | 需要电池 | 可制作声波手电筒 |
| 小孩 | 小声，容易哭 | 需要食物和水 | 感知范围扩大 |

## 3. 触控交互设计

### 3.1 移动与战斗

| 手势 | 功能 | 触觉 |
|------|------|------|
| 单指 ↑↓←→ 滑动 | 向对应方向移动 | 脚步振动反馈 |
| 双击 | 近战攻击 | 攻击振 |
| 长按 | 屏息静止（躲避声呐型怪物） | 心跳脉动（逐渐增强） |
| 覆盖整个屏幕（手掌） | 完全伏地隐藏 | 初始重振后静默 |

### 3.2 物品与交互

| 手势 | 功能 |
|------|------|
| 双指 ← 滑 | 切换武器 |
| 双指 → 滑 | 使用当前物品 |
| 双指 ↑ 滑 | 拾取地面物品 |
| 三指点击 | 发出呼救声（吸引 NPC 但可能吸引怪物） |
| 单指画圈 | 拍手（主动声呐——判断当前房间） |

### 3.3 状态界面

| 手势 | 功能 |
|------|------|
| 三指长按 | 语音播报当前状态（血量/口渴/饥饿/位置） |

## 4. 音频设计

### 4.1 音频分层

```
Layer 4 ┃ 背景音乐（仅安全区和 Boss 战出现）
Layer 3 ┃ 环境底噪（通风声、远处怪物声、建筑应力声）
Layer 2 ┃ 玩家动作音（脚步、呼吸、心跳、物品使用）
Layer 1 ┃ 核心交互音（怪物声音、NPC 声音、资源声音、回声）
```

### 4.2 3D 空间音频

- 所有怪物和 NPC 声音使用双耳空间音频渲染
- 脚步声材质系统：6 种地面材质，实时切换
- 距离衰减：近处声音（< 3m）清晰但令人紧张，中距离（3-15m）提供反应时间，远处（> 15m）作为预警

### 4.3 心率系统

玩家血量通过心率声表达：
- 100-75% HP：正常心率，几乎不被注意
- 75-50% HP：心率加快，间歇性可闻
- 50-25% HP：心率持续可闻，声音变大
- 25% 以下：心率极快且响亮，伴随耳鸣效果

**副作用**：心率声越大，越容易被怪物听到。

### 4.4 语音内容

| 内容 | 数量 |
|------|------|
| 幸存者 NPC 对话 | 8 个角色 × 15-20 句 |
| 录音带 | 20 个，每个 30-60 秒 |
| 玩家角色配音 | 呼吸、受伤、攻击等非语言音效 |
| 系统提示 | ~80 条语音提示 |

## 5. Roguelike 系统

### 5.1 死亡与重生

- 死亡后进入"结算界面"（纯音频）
- 语音播报本次存活天数、击杀数、收集录音带数
- 获得"回声点数"，用于永久升级

### 5.2 永久升级（回声天赋树）

| 升级 | 满级 | 最大效果 |
|------|------|---------|
| 敏锐听觉 | Lv.5 | 可听范围扩大 40% |
| 无声移动 | Lv.5 | 自身移动声音降低 50% |
| 坚韧心智 | Lv.3 | 心率声阈值降低（满 HP 时更不易触发） |
| 回声记忆 | Lv.3 | 已探索房间在回归时自动识别 |
| 声波感知 | Lv.4 | 可以发出超声波感知墙壁后的东西 |
| 幸存者亲和 | Lv.3 | NPC 出现概率增加 |

### 5.3 结局系统（20 个结局）

结局取决于：
- 收集到的录音带数量（揭示黑暗降临的真相）
- 成功营救的幸存者数量
- 最终区域的选择

## 6. 地图生成

### 6.1 算法

- 使用随机游走 + 细胞自动机生成房间布局
- 每个房间在生成时被赋予随机材质和大小
- 关键房间模板（安全屋、Boss 房、录音带房间）手动设计后随机放置
- 保证生成的地图中存在至少一条到达出口的可行路径

### 6.2 区域主题

| 区域 | 环境音特征 | 主要材质 | 常见怪物 |
|------|----------|---------|---------|
| 公寓楼 | 风声、远处电视雪花声 | 木板、地毯 | 呢喃者 |
| 地下车库 | 滴水声、回声长 | 混凝土 | 咔嗒兽 |
| 医院 | 心电图声、药瓶碰撞声 | 地砖 | 盲猎者 |
| 地铁隧道 | 远处列车回响 | 金属、碎石 | 尖叫行者 |
| 商场 | 破损的广播音乐 | 大理石 | 多种怪物混合 |

## 8. 无障碍
