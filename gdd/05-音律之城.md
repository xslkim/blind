# GDD-005：音律之城

> 版本：v1.0 | 日期：2026-05-12 | 状态：策划阶段

---

## 1. 游戏概述

### 1.1 核心信息

| 项目 | 内容 |
|------|------|
| 游戏名称 | 音律之城（Melody City） |
| 类型 | 音乐创作 + 城市探索 + 社交 |
| 平台 | iOS / Android |
| 技术栈 | Unity + FMOD + 自定义音乐引擎 |
| 目标受众 | 音乐爱好者、放松类游戏玩家、创作型玩家 |
| 游戏时长 | 无限（沙盒），主线约 3 小时 |
| 定价模式 | 免费 + 内购乐器/皮肤（音频效果） |

### 1.2 Elevator Pitch

> 欢迎来到永夜之城——一座没有光、只有声音的城市。在这里，你是一位街头音乐人，用指尖的旋律探索世界、结识同好、在街头即兴合奏。《音律之城》是治愈的、自由的、属于每一个人的音乐沙盒。

### 1.3 核心体验目标

- **自由创作**：不需要乐理知识，通过直觉式的触摸界面演奏出好听的旋律
- **社交共鸣**：与 NPC 和其他玩家的"音乐化身"即兴合奏，是核心快乐来源
- **发现感**：城市的每个角落有不同的音乐风格，探索即发现新的音乐语言

## 2. 核心玩法

### 2.1 游戏循环

```
在城市中行走 → 听到不同区域的音乐 → 加入演奏 → 与 NPC/玩家合奏
      │                                                    │
      └── 解锁新乐器/新曲谱/新区域 ←── 默契度/声望提升 ←──┘
```

### 2.2 核心机制

#### 2.2.1 虚拟乐器演奏系统

**触摸演奏界面**：

屏幕被虚拟划分为 3 列 × 4 行共 12 个触控区。每个区域的触觉反馈微有不同，帮助玩家用肌肉记忆定位。

**演奏模式**：

| 模式 | 触控方式 | 音乐效果 |
|------|---------|---------|
| 单音模式 | 点击一个格子 | 演奏该格子对应的音符（音高由行决定，音色由列决定） |
| 滑音模式 | 手指在格子上滑动 | 音符之间平滑过渡 |
| 和弦模式 | 多指同时按多个格子 | 同时演奏多个音符 |
| 节奏模式 | 快速连续点击 | 自动量化到最近的节拍 |
| 力度模式 | 按住 + 压力感应（3D Touch/等效） | 音量随按压力度变化 |

**可选乐器**（通过双击切换）：

| 乐器 | 音域 | 演奏风格 | 解锁方式 |
|------|------|---------|---------|
| 口琴 | 2 个八度 | 温暖、呼吸感 | 初始 |
| 木吉他 | 3 个八度 | 柔和、和弦感 | 完成教程 |
| 钢琴 | 4 个八度 | 清澈、精确 | 声望 Lv.3 |
| 电吉他 | 3 个八度 | 力量、失真 | 工业区任务 |
| Beatbox | 节奏组 | 打击乐 | 街头对战胜利 |
| 合成器 | 3 个八度 | 电子、氛围 | 隐藏任务 |
| 小提琴 | 3 个八度 | 抒情、连奏 | 公园区任务 |
| 古筝 | 3 个八度 | 东方韵味 | 旧城区任务 |

#### 2.2.2 城市区域音乐系统

| 区域 | 背景音乐风格 | BPM | 调性 | 特色乐器 |
|------|------------|-----|------|---------|
| 旧城区 | 爵士 / Blues | 60-80 | C 小调 | 萨克斯、低音提琴 |
| 工业区 | Techno / Industrial | 120-140 | E 弗里吉亚 | 合成器、鼓机 |
| 公园 | 民谣 / 自然 | 90-110 | G 大调 | 木吉他、口哨 |
| 市中心 | Pop / Funk | 100-120 | D 多利亚 | 贝斯、键盘 |
| 滨水区 | Ambient / 新世纪 | 70-90 | A 自然小调 | 钢琴、弦乐 |
| 地下街 | Hip-hop / Lo-fi | 80-95 | F 小调 | 采样、808 |
| 塔顶 | 管弦 / 史诗 | 变拍 | 全音阶 | 管弦全编制 |

每个区域的背景音乐是一个持续的、动态的伴奏轨道。玩家加入演奏时，系统自动分析玩家旋律的调性和节奏，调整背景音乐的伴奏模式与之匹配。

#### 2.2.3 合奏系统

**与 NPC 合奏**：
- 当玩家靠近一个正在演奏的 NPC 时，听到他们的演奏声
- 在 NPC 附近开始演奏即自动加入合奏
- 系统评估"默契度"：
  - 和谐度：旋律与 NPC 旋律的和声匹配度
  - 节奏契合：节奏稳定性与同步度
  - 创意加分：在 NPC 旋律基础上的创新

**默契度效果**：

| 默契度 | 解锁内容 |
|--------|---------|
| C（及格） | 解锁 NPC 基础对话 |
| B（良好） | NPC 分享一段曲谱 |
| A（优秀） | 解锁该 NPC 的支线故事 |
| S（完美） | NPC 成为常驻合奏伙伴，可随时呼叫 |

**与玩家合奏（社交功能）**：
- 玩家的"音乐化身"会在城市中留下演奏录音
- 其他玩家经过时听到录音并可以叠加演奏
- 录音随时间衰减（24 小时后音量降低，72 小时后消失）
- 每周"音乐节"事件：全球玩家同时演奏，生成一段集体创作的城市主题曲

## 3. 触控交互设计

### 3.1 演奏手势

| 手势 | 功能 |
|------|------|
| 单指点击网格区 | 演奏单个音符 |
| 单指在网格区滑动 | 滑音 |
| 多指同时点击 | 和弦 |
| 在网格区上方快速横扫 | 琶音（快速依次演奏一行） |
| 双击任意位置 | 切换乐器（循环） |
| 长按网格区某格 | 持续音（音量随压力变化） |
| 摇晃手机 | 打击乐（沙锤/铃鼓音效） |
| 三指点击 | 开始/停止录音（录制演奏片段） |

### 3.2 移动/导航手势

| 手势 | 功能 |
|------|------|
| 网格区外 ↑↓←→ 滑动 | 在城市中向对应方向移动 |
| 网格区外双指缩放 | 切换移动速度（步行/跑步） |
| 覆盖屏幕（手掌） | 停下，静听环境 |

### 3.3 触觉反馈

| 事件 | 触觉 |
|------|------|
| 按下一个音符 | 微振（强度与力度对应） |
| 进入新的城市区域 | 区域特征振动模式（旧城区=柔和脉动 / 工业区=强力节拍）
| 合奏达到高默契度 | 全身振动共鸣（模拟音乐共鸣感） |
| NPC 靠近 | 渐强脉动 |
| 发现隐藏曲谱 | 惊喜三连振 |

## 4. 音频设计

### 4.1 动态音乐引擎

**核心引擎功能**：
1. **调性分析**：实时分析玩家演奏的音符，推断当前调性和情绪
2. **自动伴奏**：根据推断的调性生成伴奏轨道
3. **风格过渡**：在不同城市区域之间移动时，伴奏以 DJ 混音方式过渡
4. **合奏混音**：多个演奏者的音频自动混音并空间化

**技术方案**：
- FMOD Studio 用于背景音乐的分层播放和参数过渡
- 自定义音符分析器（FFT + 音高检测）
- 自动伴奏使用预录制的和弦进行的变调拼贴（Granular Synthesis）
- 所有区域伴奏轨道提前录制为多轨道 Stem，运行时混合

### 4.2 NPC 音乐人设计

| NPC | 演奏乐器 | 音乐风格 | 性格 |
|-----|---------|---------|------|
| 老陈 | 二胡 | 传统民乐 | 沧桑、怀旧 |
| Mia | 电子合成器 | Synthwave | 酷、科技感 |
| 大狗 | 贝斯 | Funk | 热情、话多 |
| 小雨 | 小提琴 | 古典/跨界 | 害羞但技艺高超 |
| KAI | Beatbox | Hip-hop | 自信、有些自负 |
| 奶奶 | 钢琴 | 古典 | 温暖、有故事 |

每个 NPC 有 15-20 段对话（配音），包含他们的音乐故事和人生故事。

### 4.3 音乐节系统

**每周音乐节（线上事件）**：
- 每周六 20:00-21:00（服务器时间），所有在线玩家进入"音乐节"模式
- 所有玩家演奏自动记录，合并为城市的"集体音轨"
- 音乐节结束后，生成一段 3 分钟的"本周城市之声"，所有参与者可下载
- 连续参加可获得"音乐节常客"称号和解锁限定乐器

## 5. 叙事与世界观

### 5.1 世界观

> 不知从什么时候起，这座城市陷入了永恒的夜晚。不是恐怖的黑——而是一种安宁的、让人安静下来倾听的黑暗。人们发现，失去了视觉的干扰后，音乐变得更加纯粹。城市的经济、社交、文化全部围绕音乐重新运转。而你，是一个新来的街头音乐人，带着你的乐器和好奇心。

### 5.2 主线

主线是轻量级的——核心驱动力是探索和音乐创作，而非剧情。

| 阶段 | 目标 | 时长 |
|------|------|------|
| 初来乍到 | 学习演奏、探索旧城区 | 30min |
| 声名鹊起 | 与至少 3 个区域的 NPC 合奏达 A 级 | 1h |
| 城市之心 | 在全部 7 个区域演奏并留下录音 | 1h |
| 音乐节传奇 | 参加音乐节并达到前 10% 贡献 | 30min |

### 5.3 隐藏内容

- 散布在城市各处的"失落的旋律"（收集后解锁特殊曲谱）
- 只有在凌晨 3-5 点（服务器时间）才会出现的"午夜音乐家"
- 城市塔顶的"沉默之厅"——唯一没有音乐的场所，在这里可以听到所有区域音乐的混响回声

## 6. 社交系统

### 6.1 异步社交

- 玩家演奏录音的"音乐化身"自然散落在城市中
- 听到心仪的街头演奏可以"点赞"，对方上线后收到通知
- "合奏邀请"：在一段录音上叠加演奏后，原主人收到合奏邀请

### 6.2 实时社交（可选）

- 可以加入最多 4 人的实时合奏房间
- 实时音频流延迟目标 < 100ms
- 合奏房间有文字聊天转语音功能
