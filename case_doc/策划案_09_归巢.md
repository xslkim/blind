# 策划案 09：《归巢》—— 鸟类导航模拟

> **共通原则**：全程黑屏无 UI 像素输出（无障碍与可选辅助另案）；交互依赖触控、空间音频、音乐、音效与 NPC 配音；建议耳机校准与触觉反馈。

**一句话**：扮演迁徙中的鸟，仅凭太阳偏振感（以音效象征）、风声与族群呼叫找路。

**目标用户**：自然主题与家庭友好玩家。

**核心玩法**：朝某一方向快速滑动表示奋力飞行，耗能以呼吸声加重提示；点击回应族群呼叫维持「群体 buff」；暴风雨关卡需听闪电延迟估距离后短滑躲避（以风声突变代替画面）。

**触控设计**：环形滑动改变「朝向」；中心长按滑翔省能。

**音频设计**：多普勒效应用于判断同伴相对速度；成功归巢时雏鸟叽叽渐近。

**叙事**：无人类台词，仅自然与族群，结局以季节更替环境音表示。

**技术要点**：教育向可选事实解说音轨（博物学科普）。

**差异化**：「非人主角」的感知逻辑，教育意义与玩法合一。
