# 策划案 04：《茧》—— 触觉与虫鸣的心理惊悚

> **共通原则**：全程黑屏无 UI 像素输出（无障碍与可选辅助另案）；交互依赖触控、空间音频、音乐、音效与 NPC 配音；建议耳机校准与触觉反馈。

**一句话**：你困在看不见的茧中，靠指尖「剥落」丝线与虫声判断自己还剩多少理智。

**目标用户**：能接受轻度惊悚的成年玩家；夜间耳机体验。

**核心玩法**：多指向外划开模拟撕茧，音效提示层数；停顿时蚕噬声逼近，需规律轻点「镇定呼吸」；剧烈滑动会加速「破裂」也加速「暴露」给未知存在。

**触控设计**：轻点、重点、交替点形成不同「呼吸曲线」识别。

**音频设计**：ASMR 与不安低频交替；远处孩童笑声是否为幻听由关卡随机决定，增加不可复玩惊悚。

**叙事**：极简，靠碎片化耳语与新闻广播切片暗示实验体设定。

**技术要点**：内容警告与亮度锁定；心理健康资源外链（可选）。

**差异化**：把「自我伤害式交互」（撕扯）与风险机制绑定，道德张力由音效承担。
