# 策划案 07：《骨牌》—— 只听得见的连锁反应谜题

> **共通原则**：全程黑屏无 UI 像素输出（无障碍与可选辅助另案）；交互依赖触控、空间音频、音乐、音效与 NPC 配音；建议耳机校准与触觉反馈。

**一句话**：摆放「听得见」的骨牌，推倒时须达成指定节奏型。

**目标用户**：解谜与节奏玩家。

**核心玩法**：点击放置「高音」「低音」「休止」三类隐形骨牌（放置点用极短不同音效确认，无画面）；滑动手指链式推倒；关卡目标为 NPC 敲击一段节奏，玩家复现连锁时长与间隔。

**触控设计**：撤销为两指左滑；重做三指点击。

**音频设计**：材质 pack（木、瓷、金属）改变泛音；失败时温和「塌成一团」的喜剧化音效降低挫败。

**叙事**：轻量，以街头艺人 NPC 串场。

**技术要点**：物理音频可简化为规则合成，降低算力。

**差异化**：谜题目标是「节奏型」而非「图案」。
