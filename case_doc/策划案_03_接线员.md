# 策划案 03：《接线员》—— 仅声波的冷战谍报

> **共通原则**：全程黑屏无 UI 像素输出（无障碍与可选辅助另案）；交互依赖触控、空间音频、音乐、音效与 NPC 配音；建议耳机校准与触觉反馈。

**一句话**：你是黑屋中的无线电接线员，靠旋钮与摩斯节奏分辨敌我。

**目标用户**：历史谍战题材爱好者；强代入感。

**核心玩法**：单指圆周滑动模拟调频，双指缩放模拟滤波带宽；语音情报夹杂噪声，玩家需「锁定」清晰频段后长按录音提交；假信号以相似人声陷害，需比对背景环境音（列车 vs 咖啡馆）。

**触控设计**：刻度震动模拟机械旋钮段落感；快速双击紧急上报。

**音频设计**：多语种低语、发报机、管风琴远处泄漏音；NPC 上司只以声音出现，情绪影响任务难度。

**叙事**：一夜之间的叛变疑云，真结局藏在最后一条被干扰的半句人声中。

**技术要点**：动态范围控制避免听力疲劳；字幕可选（默认关闭，仅无障碍）。

**差异化**：「辨伪人声」而非辨色辨图，强调听觉微差。
