# 策划案 01：《盲径》—— 听觉寻路与生存

> **共通原则**：全程黑屏无 UI 像素输出（无障碍与可选辅助另案）；交互依赖触控、空间音频、音乐、音效与 NPC 配音；建议耳机校准与触觉反馈。

**一句话**：玩家在完全黑暗中仅凭双耳与指尖「摸」出安全路径，逃离追逐声。

**目标用户**：喜欢紧张感与耳机沉浸的玩家；通勤短时游玩。

**核心玩法**：单指滑动模拟「伸手探路」，系统以材质音效反馈（金属栏杆、布帘、积水）；双指捏合表示「屏息」降低被听觉猎手发现的概率；长按触发「投掷小石子」引开敌人，有次数限制。

**触控设计**：滑动手速与压力（若设备支持）影响脚步轻重；边缘短震提示「碰壁」。

**音频设计**：3D 空间化脚步与回声；远处 NPC 低语作为叙事碎片；心跳与呼吸随危险等级混音。

**叙事**：废弃设施中的无名逃亡者，通过录音机残片拼凑真相。

**技术要点**：头部相关传递函数（HRTF）优先；低延迟触控采样；可选「视觉全无」认证模式（锁屏纯音频需另行评估）。

**差异化**：「材质听觉地图」作为核心记忆负担，而非传统迷宫背板。
