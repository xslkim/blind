# 策划案 02：《潮信》—— 潮汐与时间的音乐解谜

> **共通原则**：全程黑屏无 UI 像素输出（无障碍与可选辅助另案）；交互依赖触控、空间音频、音乐、音效与 NPC 配音；建议耳机校准与触觉反馈。

**一句话**：黑屏下以节奏与海浪相位解开古老灯塔的秘密。

**目标用户**：音游与叙事向玩家；偏放松但需专注。

**核心玩法**：NPC 守塔人哼唱固定动机，玩家以点击、长按、滑扫复现旋律片段；错误时潮水声「淹没」进度一节；潮汐涨落以低频嗡鸣相位变化提示，玩家需「听出」安全窗口。

**触控设计**：点击为鼓点；长按为持续音；滑扫为琶音；多指同时按模拟和弦（关卡渐进解锁）。

**音频设计**：实时混音层数随解谜成功叠加；环境音（鸥鸣、绳索、齿轮）作为误导与线索并存。

**叙事**：三代守塔人家书式独白，结局由玩家最后一次「合奏」选择分支。

**技术要点**：节奏判定宽容度可调；可提供「节拍器滴答」辅助音轨开关。

**差异化**：谜题即音乐结构本身，无谱面 UI，靠记忆与耳力。
