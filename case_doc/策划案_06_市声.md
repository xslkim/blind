# 策划案 06：《市声》—— 黑屏里的城市策展人

> **共通原则**：全程黑屏无 UI 像素输出（无障碍与可选辅助另案）；交互依赖触控、空间音频、音乐、音效与 NPC 配音；建议耳机校准与触觉反馈。

**一句话**：收集并混音一条看不见的城市声景，交给看不见的委托人评判。

**目标用户**：实验艺术与放松玩家。

**核心玩法**：单指在不同屏幕区域（无画面，靠初始教学语音分区）录制环境采样；双指交叉淡入淡出；三指旋转调整「时间层」（晨/午/夜）环境底噪。

**触控设计**：分区记忆依赖教学后玩家自建心理地图，可开启「角落语音坐标」辅助。

**音频设计**：NPC 委托人各有「声学品味」任务目标（更吵、更空、更旧等抽象描述）。

**叙事**：委托人实为已故录音师的分身，终章揭示玩家也在被收录。

**技术要点**：本地 DSP 性能与循环无缝剪辑；分享导出为音频文件而非视频。

**差异化**：「抽象美学目标」由 NPC 口头描述驱动，非传统分数 UI。
