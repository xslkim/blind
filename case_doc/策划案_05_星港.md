# 策划案 05：《星港》—— 无画面的太空停靠

> **共通原则**：全程黑屏无 UI 像素输出（无障碍与可选辅助另案）；交互依赖触控、空间音频、音乐、音效与 NPC 配音；建议耳机校准与触觉反馈。

**一句话**：只听引擎与导航 AI，把货船泊入旋转中的盲港。

**目标用户**：科幻硬核与模拟类玩家。

**核心玩法**：倾斜设备（若允许传感器）或双指上下滑模拟推力；左右滑偏航；AI 以口令报相对距离与相对角速度；玩家需在口令间隙短按确认「点火/关停」。

**触控设计**：三指上划紧急中止；单指连点切换「监听频道」（货舱 / 塔台 / 机械故障）。

**音频设计**：金属应力呻吟、燃料泵脉冲、对接成功时的真空寂静一秒「奖励静音」。

**叙事**：AI 逐渐展现人类情感残留，停靠任务与伦理抉择通过语音选项（向左滑同意 / 向右滑拒绝）完成。

**技术要点**：若坚持纯触控，可提供「虚拟倾斜」替代陀螺仪。

**差异化**：「寂静作为正反馈」的关卡设计，与通常爆炸音效相反。
