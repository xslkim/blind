# 策划案 08：《药引》—— 中医柜前的听觉配方

> **共通原则**：全程黑屏无 UI 像素输出（无障碍与可选辅助另案）；交互依赖触控、空间音频、音乐、音效与 NPC 配音；建议耳机校准与触觉反馈。

**一句话**：老药铺黑屋内，只听抽屉推拉声与药性描述，配出救命一方。

**目标用户**：国风文化向；偏好慢节奏。

**核心玩法**：九宫格抽屉以「行列语音坐标」称呼（如「丙三」）；拖拽抽屉距离产生不同木质摩擦音提示深浅；错配时 NPC 咳嗽加重；正确时药碾声渐起。

**触控设计**：长按嗅闻（实为播放药味相关环境联想音：晒场、雨苔）；双指搓动模拟碾药完成关卡。

**音频设计**：古琴与环境音景分层；NPC 老先生气声与停顿作为线索。

**叙事**：师徒传承与战乱中一封迟到家书。

**技术要点**：文化顾问校对药名读音；避免伪科学疗效宣称（Disclaimer）。

**差异化**：「嗅觉通感」完全用声音象征，形成独特文化记忆点。
