# 策划案 10：《证词》—— 法庭只存在于耳边

> **共通原则**：全程黑屏无 UI 像素输出（无障碍与可选辅助另案）；交互依赖触控、空间音频、音乐、音效与 NPC 配音；建议耳机校准与触觉反馈。

**一句话**：黑屏法庭中，你通过打断、追问与物件呈示（以声音象征）找出证词矛盾。

**目标用户**：推理与播客爱好者。

**核心玩法**：证人与律师对话流自动播放；玩家在听到矛盾点时单指点击「反对」；双指滑动翻阅「听觉物证」（录音、街道声纹）；长按迫使证人重复上一句，语音情绪变化可能暴露谎言。

**触控设计**：时间轴无 UI，以「捏合两指」缩放复述片段长度（近景听呼吸 / 远景听全场）。

**音频设计**：空间混响区分证人席、陪审团窃窃私语、法官锤声位置。

**叙事**：多结局；NPC 全语音驱动，关键证据为环境声细节（钟声次数、远处狗吠品种）。

**技术要点**：大量本地化配音成本；脚本分支工具链重要。

**差异化**：「听觉物件」作为证据类型，拓展推理游戏边界。
